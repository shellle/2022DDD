# ・心理的障壁に定するアプローチ

他人とのコミュニケーションの際に
常に相手の状況を注意，意識しすぎるがために情報共有が滞ってしまうことがある．

そこで僕らは，各メンバーが休暇中であることを設定できる機能を搭載することによって，
メッセージの送信者は相手の状況を気にせずに送信できる，休暇時間配慮送信の機能を用いることで
送信する際の心理的障壁を取り除けるのではと考えた．
